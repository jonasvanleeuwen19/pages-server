Quick start guide

1. Create a branch named {preferred_branch}.

2. Add your static site to the repository root or use folders like public, dist, or site.

3. Enable automatic updates in the repository settings page.

4. Visit your site at https://USERNAME.{main_domain}/REPO.

5. Optional: add a .pages file: {{"main-site": true, "domains": ["example.com"]}}

If main-site is enabled, your repo also becomes available at https://USERNAME.{main_domain}.
